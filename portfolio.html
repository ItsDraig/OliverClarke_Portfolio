001 Full-Stack
Let's Cook
Let's Cook helps you make the most of what's already in your kitchen by tracking your on-hand ingredients and their expiration dates to reduce food waste. It then generates creative, delicious recipe ideas tailored to what you have — so you can use up ingredients in time and cook with confidence.
React Native Node.js PostgreSQL OpenAI Supabase Vercel
+
The Details
Let's Cook is a full-stack kitchen management app that eliminates the daily "what do I make with this?" problem. Users maintain a live pantry — adding ingredients, quantities, and expiration dates — and the app surfaces what's at risk of going to waste. From there, a single click hits the OpenAI API with a carefully engineered prompt that accounts for the user's available ingredients, dietary preferences, and portion size, returning a structured recipe with ingredients, steps, and estimated cook time.

The ingredient database is backed by the Spoonacular API, which provides autocomplete, nutritional data, and unit normalization — so "half a block of firm tofu" and "200g tofu" resolve to the same pantry item. Authentication and real-time pantry sync are handled through Supabase, giving each user a persistent, cross-device kitchen without building a custom auth system. The frontend is built in React Native with a component architecture designed around the pantry state as the single source of truth — recipe generation, expiry alerts, and dietary filtering all derive from it.

002 API / Backend Open Source
Project Draftr
Utilizing Riot Games official API. Project Draftr is a tool for analyzing League of Legends team compositions. Including information about their strengths, weaknesses, and how well they would perform into other compositions.
Python FastAPI Docker AWS
+
The Details
Project Draftr is a full-stack League of Legends tool with backend/data science focus that evaluates five-champion team compositions in real time, assigning them a composition archetype and a letter grade from D to S. Users select their five picks and the app immediately scores the draft across three dimensions: aggregate winrate (pulled live from the Riot Games API by rank tier), pairwise synergy between champions, and composition coherence — how cleanly the draft fits a recognized playstyle.
